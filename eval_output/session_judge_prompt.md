# GER-RAG セッション適応性 関連度判定

各シナリオで、異なるトピックの交互クエリを大量に実行した前後で
観測クエリの検索結果がどう変わったかを判定してください。

## 判定基準

| スコア | 意味 |
|--------|------|
| 0 | 無関係（クエリと接点なし） |
| 1 | やや関連（話題がかすっている程度） |
| 2 | 関連あり（クエリのトピックに明確に関連） |
| 3 | 高い関連性（クエリの意図に直接合致） |

## 回答フォーマット

```json
{
  "S1_before": [3, 2, 1, ...],
  "S1_after": [3, 3, 2, ...],
  "S2_before": [...]
}
```
