  このMCPは、AIの長期記憶用に作りたいです。
  
  1. 誰がクライアント？ Claude Desktop? 自作エージェント? 他のMCPクライアント?　→　コーディングエージェントを主眼に置くが、自作エージェントなどからも呼べるように。
  2. GER-RAGの重力・創発性をMCPでどう活かす？ 単なるRAGツールだともったいない　→　エージェント自身の発言もMCP経由で記録できるようにする。双方向利用できる、「自分の外部記憶」。ロングコンテキスト時のコンパクティングをここに記録できると面白いかも。
  3. stateの扱い — MCPセッション中にクエリするたびに重力が蓄積する。これを「会話が深まるほど検索が賢くなる」体験にできるか　→　できるだけ保持したい。
  4. リソース — MCPのResourcesとして知識グラフの状態を公開する？　→　公開する。
  5. プロンプト — よく使うクエリパターンをMCPプロンプトとして定義する？　→　定義する。
