# 🌱 はじめての GaOTTT (4/6) — 最初の会話

接続できたので、いよいよ Claude に「覚えてもらう」「思い出してもらう」を試します。

← [前: Connect](Tutorial-03-Connect-To-Claude.md) ｜ → [次: Everyday Use](Tutorial-05-Everyday-Use.md)

---

## 試 1: 何かを覚えてもらう

Claude Desktop に、こう話しかけてみてください:

```
私が好きな食べ物は焼き鳥の皮です。これを記憶しておいてもらえる？
```

### Claude の反応

Claude は `remember` ツールを呼んで、こう返してきます:

```
覚えました！「私が好きな食べ物は焼き鳥の皮」を保存しました。
ID: 7a3c9b2d-...
```

ID が表示されれば成功 ✅。

---

## 試 2: 思い出してもらう

Claude Desktop で、**新しい会話** を始めてください（左上の「+」で新規セッション）。

新しい会話で:

```
私が好きな食べ物について、覚えていることはある？
```

### Claude の反応

Claude は `recall` ツールを呼んで、先ほど保存した記憶を見つけてきます:

```
記憶を確認しました。あなたが好きな食べ物は「焼き鳥の皮」だと記録されています。
```

**新しい会話なのに思い出せた** これが GaOTTT の威力 🎉。

---

## 試 3: 「人格」を宣言してみる

Claude にあなたの価値観や好みを覚えてもらうと、次のセッションでも引き継いでくれます。

```
declare_value で、私の価値観を 1 つ覚えてほしい:
「迷ったら、より長く面白いと思える方を選ぶ」
```

別のセッションで:

```
inherit_persona を呼んで、私の価値観を確認してみて
```

→ Claude が宣言した value を読み上げてくれます。**これがセッションをまたぐ「人格保存」** です。

---

## 試 4: タスクを管理してもらう

```
commit で、タスクを 1 つ作って:
「Tutorial-05 を読む」
```

別のセッションで:

```
reflect で aspect="tasks_todo" を確認して、今やるべきタスクを教えて
```

→ さっき作ったタスクが浮上します。

完了したら:

```
そのタスクを complete して、outcome は「Tutorial 全部読んだ」で
```

→ タスクは完了として記録され、**「成し遂げたことの年表」** が積み重なっていきます。

---

## 試 5: 普段の会話の中で自然に使ってもらう

Claude に、こう伝えておくと良いです（毎セッション最初に）:

```
このセッションでは、重要な気づきや決定があったら GaOTTT memory に
remember で保存しておいて。私が「前にこう言ってたよね」と確認できる
ように。
```

これだけで、**Claude が自発的にメモを取ってくれる** ようになります。

---

## 何が起きてるか（雰囲気だけ）

- 保存した記憶は、あなたのパソコンの `~/.local/share/gaottt/` に保存されています
  - Mac/Linux なら `/Users/あなたの名前/.local/share/ger-rag/`
  - Windows なら `C:\Users\あなたの名前\AppData\Local\ger-rag\`
- セッションをまたいで、別の Claude も同じ場所を読みます
- たくさん使うほど、関連する記憶どうしが**重力で引き寄せ合う**ようになります（`recall` するたびに引力が増します）
- 不要な記憶は自然に薄れていき、ほっとくと消えます（**忘れる勇気** が物理的に組み込まれています）
