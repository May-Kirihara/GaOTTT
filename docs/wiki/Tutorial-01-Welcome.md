GaOTTT にはじめて触れる方への、ゆっくり進むガイドです。Python や開発をしたことがなくても大丈夫。**LLM が好きで、もっと深く付き合いたい人** が対象です。

## これ、何が嬉しいの？

**LLM には記憶がありません。** 毎回のセッションで、あなたが誰なのか、前回何を話したか、好きなものは何か、全部忘れてしまいます。

GaOTTT をつなげると、LLM は **過去の会話を覚えていられる** ようになります。

### こんなことができるようになります

- 🌟 「前回話した〇〇の続きから」と言えば、LLM が思い出してくれる
- 🌟 あなたの好み・口調・大切にしている価値観を覚えていてくれる
- 🌟 セッションをまたいで「進行中のタスク」を管理できる
- 🌟 何度も思い出した記憶ほど、強く残るようになる（**重力で記憶が育つ**）
- 🌟 関係ないと思っていた記憶同士が、ある日突然つながる発見がある

### イメージとしては

> LLM が **あなた専用のジャーナル + メモ帳 + 内省パートナー** を持つようになる感覚。
> 書いた内容が時間とともに重みを持ち、似た想いが互いに引き寄せ合っていく。

公式の説明はちょっと難しいですが、要するに **「LLM にあなた専用の長期記憶をつけるツール」** です。

## 必要なもの

事前にこれだけ用意してください:

### 必須

- ✅ **パソコン** (Mac / Windows / Linux)
  - 最近の機種なら大丈夫です（メモリ 4GB 以上推奨）
  - **約 4GB の空き容量**（モデルファイルが約 1.2GB + データ）

- ✅ **LLM Desktop アプリ**
  - これが GaOTTT と接続する相手です

- ✅ **インターネット接続**（初回のみ）
  - モデルファイルをダウンロードするのに必要
  - 1 度ダウンロードしたら、以降はオフラインでも動きます

### あると便利

- 📝 **ターミナル**（コマンドを打つ画面）に少しでも触れたことがある
  - ほぼ初めてでも大丈夫。次のページで丁寧に説明します
  - Mac の方: 「ターミナル.app」
  - Windows の方: 「PowerShell」

### 必要ないもの

- ❌ プログラミングの知識
- ❌ Python の経験
- ❌ Git の知識（なくても進めるよう手順書きます）
